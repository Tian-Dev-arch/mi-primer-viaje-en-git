# mi-primer-viaje-en-git

Espacio para dar mis primeros pasos en el control de versiones. 

Este es mi primer contacto con GitHub para el tema de control de versiones de los programas que se crearan de ahora en adelante. **¡Vamos con toda!** 🔥

---

### 🚀 Objetivos de aprendizaje:
- **Lenguaje:** Se usara el lenguaje de Python para cada proyecto.
- **Control de versiones:** Entender el flujo de trabajo de Git y GitHub.
- **Testing:** Aprender a diseñar y escribir pruebas automatizadas.
- **Buenas prácticas:** Estructurar código limpio y mantenible.
